# Amazon Sentiment Analysis

## Project Overview
This project applies Natural Language Processing (NLP) techniques to analyze customer sentiment from Amazon product reviews. The goal was to classify reviews as positive, negative, or neutral and identify overall customer opinion patterns.

## Objectives
- Clean and preprocess text data
- Perform sentiment analysis using VADER
- Visualize sentiment distribution
- Extract insights from customer feedback

## Tools & Technologies
- Python
- Pandas
- NLTK
- Matplotlib
- Seaborn
- Jupyter Notebook

## Process
1. Data loading and exploration
2. Text preprocessing
3. Sentiment scoring using VADER
4. Classification of reviews
5. Visualization of sentiment trends

## Results
- Majority of reviews were classified as positive
- Negative reviews highlighted areas for product improvement
- Sentiment analysis provided insights into customer satisfaction

- Natural Language Processing (NLP)

- Sentiment Analysis
